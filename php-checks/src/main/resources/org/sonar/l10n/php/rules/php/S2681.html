单行块可以省略圆括号，例如使用if语句或for循环，但是这样做可能会误导和引发错误。

当一行代码块后面的行出现空白，表示有意将这些行包含在代码块中，但是没有圆括号意味着这些行将无条件地执行一次时，这条规则就会产生问题。

不合规的代码示例

if ($condition)
  firstActionInBlock();
  secondAction();  // 不合规；无条件执行
thirdAction();

if($condition) firstActionInBlock(); secondAction();  // 不合规；无条件执行secondAction

if($condition) firstActionInBlock();  // 不合规
  secondAction();  //无条件执行

$str = null;
for ($i = 0; $i < count($array); $i++)
  $str = $array[$i];
  doTheThing($str);  // 不合规；仅在最后一个数组元素上执行

合规的解决方案

if ($condition) {
  firstActionInBlock();
  secondAction();
}
thirdAction();

if($condition) { firstActionInBlock(); secondAction(); }

if($condition) {
  firstActionInBlock();
  secondAction();
}

$str = null;
for ($i = 0; $i < count($array); $i++) {
  $str = $array[$i];
  doTheThing($str);
}

  • CERT, EXP52-J. - Use braces for the body of an if, for, or while statement
