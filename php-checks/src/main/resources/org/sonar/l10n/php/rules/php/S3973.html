如果没有封闭的圆括号，紧接在条件句之后的行就是有条件执行的行。根据惯例和良好的实践，这些行都是缩进的。 在没有圆括号和缩进的情况下，原始程序员的意图是完全不清楚的，可能实际上并没有执行什么。此外，这样的代码很可能让维护人员感到困惑。

不合规的代码示例

if ($x > 0)  // 不合规
doTheThing();
doTheOtherThing();

foo();

合规的解决方案

if ($x > 0) {
  doTheThing();
  doTheOtherThing();
}

foo();

或者

if ($x > 0)
    doTheThing();
doTheOtherThing();

foo();
